## 🏗️ System Architecture

```
User Query
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│  Nginx Reverse Proxy  ─  Rate Limiting  ─  SSL Termination    │
└─────────────────────────────────────────────────────────────────┘
    │                                         │
    ▼                                         ▼
FastAPI Backend                     Serverless Edge (Netlify/Vercel/CF Workers)
    │                                         │
    ▼                                         ▼
Redis Cache ◄────────────────► Embedding Model (MiniLM / OpenAI)
    │                                         │
    ▼                                         ▼
┌──────────────────────────────────────────────────────────────────┐
│         Vector Store: Pinecone / Chroma / HNSWLib / FAISS       │
│                    Similarity Search + Re-ranking                │
└──────────────────────────────────────────────────────────────────┘
    │
    ▼
Context Window Assembly  ─  LLM (GPT-4o / Claude / Mistral)
    │
    ▼
Streaming Response  ─  Web Widget / WhatsApp / Telegram
```

## 🚀 Featured Projects

### 🤖 RAG Chatbot Platform
> Production-ready chatbot system deployed at scale

- Built **5+ context-aware RAG chatbots** using embeddings and vector search
- Integrated with **WhatsApp Business API** and embeddable web widgets
- Served via **Nginx reverse proxy** with Redis caching for low latency
- Deployed on **Netlify with serverless functions** for backend logic
- **Stack:** LangChain · Pinecone · FastAPI · Nginx · Redis · Netlify

---

### 🧠 Custom RAG Pipeline
> High-performance retrieval-augmented generation from scratch

- Custom **chunking strategies** + MiniLM embeddings for dense retrieval
- **HNSWLib** vector store with HNSW approximate nearest-neighbor search
- Configurable **re-ranking** and Maximum Marginal Relevance (MMR) for diversity
- REST API exposed via FastAPI, reverse-proxied with **Nginx**
- **Stack:** HuggingFace · HNSWLib · FastAPI · Docker · Python

---

### 🌐 AI Web Assistant
> Embeddable AI widget with edge-native serverless backend

- Lightweight **chatbot widget** that embeds on any website
- **Cloudflare Workers / Netlify Functions** for sub-100ms cold starts at the edge
- Intelligent query routing — answers, redirects, and escalation handling
- Streaming responses via **WebSocket / Server-Sent Events**
- **Stack:** React · Cloudflare Workers · OpenAI · Tailwind · Next.js
